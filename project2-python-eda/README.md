# 📊 French Labour Market EDA — Python Project

**Author:** Salwa Dhouibi  
**Tools:** Python · pandas · matplotlib · seaborn · Jupyter Notebook  
**Dataset:** Public data from [data.gouv.fr](https://www.data.gouv.fr) — French unemployment statistics  
**Level:** Intermediate

---

## 🎯 Objective

Perform an end-to-end Exploratory Data Analysis (EDA) on French unemployment trends.  
Combine economics expertise with data analysis skills to extract meaningful insights.

---

## 📊 Questions Explored

1. How has the unemployment rate evolved in France from 2010 to 2023?
2. Which age group is most affected by unemployment?
3. How does unemployment differ between men and women?
4. What is the impact of education level on employment?
5. Which French regions have the highest unemployment rates?
6. How does youth unemployment (15-24) compare to the national average?

## 💡 Key Findings

- Youth unemployment (15-24) consistently 2-3x higher than the national average
- Women and men show similar overall rates but structural differences by sector
- Regions with lower education levels show persistently higher unemployment
- Post-COVID recovery visible in the 2021-2023 data
